# Brand Guidelines: Blessed Anthony Etim — Personal Website

## Brand Identity

- **Brand Name:** Blessed Anthony Etim
- **Wordmark:** "Blessed Anthony Etim" set in the **primary heading font**, displayed in the navigation bar and footer
- **Brand Positioning:** A modern, credible, tech-forward legal personal brand — **professional without being stiff, intelligent without being cold**

## Typography

### Fonts

- **Primary Font:** Creato Display
- **Weights in use:** Regular **(400)** and Bold **(700)** only
- **Secondary / Body Font:** Inter *(fallback for long-form body text)*

### Typography Rules

- **Line height** for body text: `1.75`
- **Line height** for headings: `1.2`
- **Letter spacing** for H1 and H2: `-0.02em` *(tight, modern feel)*

---

## Spacing & Layout

- **Base spacing unit:** `8px`
- **Section padding** (top and bottom): `100px` desktop / `60px` mobile
- **Maximum content width:** `1200px`, centered
- **Card inner padding:** `32px`
- **Border radius** on cards and buttons: `12px`
- **Border radius** on input fields: `8px`

---

## Buttons

### Primary Button
- Background: **Electric Indigo** `#4F46E5`
- Text: **Pure White** `#FFFFFF`
- Font weight: **Bold**
- Height: `48px`
- Horizontal padding: `28px`
- Border radius: `12px`

### Secondary Button
- Background: **Transparent**
- Border: `1.5px` solid **Electric Indigo** `#4F46E5`
- Text color: **Electric Indigo** `#4F46E5`
- Same sizing as Primary Button

### Hover State
- Primary button deepens to **Deep Indigo** `#3730A3`
- Subtle upward lift: `translateY(-2px)`
- Soft drop shadow on hover

### Ghost / Text Button
- No border, no background
- **Accent color text** with underline on hover only

---

## Iconography

- Style: **Clean, minimal line icons**
- Recommended libraries: **Lucide Icons** or **Phosphor Icons**
- **Standard icon size:** `24px`
- **Inline icon size:** `20px`
- **Light mode color:** Deep Charcoal `#1A1A2E`
- **Dark mode color:** Soft White `#F0F0EC`
- **Active / highlighted state:** Electric Indigo `#4F46E5` only

---

## Imagery & Visuals

- **Photo style:** Clean, well-lit, professional — neutral or dark background preferred
- **No busy or decorative illustrations** — keep visuals minimal and purposeful
- **Decorative elements:** Abstract and geometric, using brand colors only
- **Feature image border radius:** `16px`
- **Profile / avatar border radius:** `50%` (fully circular)

---

## Dark Mode Rules

- Background → **Near Black** `#0F0F1A`
- Cards and surfaces → **Dark Navy** `#1A1A2E`
- All text → **Soft White** `#F0F0EC`
- Accent color remains **Electric Indigo** `#4F46E5` *(vivid enough for dark backgrounds)*
- Borders and dividers → **White at 10% opacity** `#FFFFFF1A`

---

## Voice & Micro-Copy Style

- **Headlines:** Bold, direct, no filler words
- **Button labels:** Action-first
  - *"Read Article"*
  - *"Download CV"*
  - *"Get In Touch"*
  - *"Subscribe"*
- **Error messages:** Calm and helpful — never technical or alarming
- **Empty states:** Encouraging
  - *"No articles yet — check back soon."*
- **Newsletter CTA:** *"Get insights on technology law delivered to your inbox."*

---

## Do's and Don'ts

### ✅ Do
- Use **Creato Display Bold** for all headings and key statements
- Keep layouts **spacious and well-padded**
- Use **Electric Indigo** for emphasis and interaction only
- Maintain **consistency** in border radius, spacing, and type scale across all pages

### ❌ Don't
- Don't use more than **two font families** anywhere on the site
- Don't overuse the **accent color** — it loses impact if used everywhere
- Don't use **busy backgrounds**, gradients, or decorative patterns
- Don't use **generic Lorem Ipsum** — all placeholder content must be contextually relevant
